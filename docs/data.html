Data Sources and Downloads

Primary data sources and processed datasets

Primary Data Sources

Shooting Victim Data

Geocoded shooting incidents in Philadelphia, 2015-present. Includes victim demographics, injury severity, and outcome information.

Records: 17,383 (after cleaning)

Source: City of Philadelphia

OpenDataPhilly - Shooting Victims

Trauma Center Registry

Accredited trauma centers in Pennsylvania with designation levels and specializations (Adult, Pediatric).

Level I Adult Centers: 4

Source: Pennsylvania Trauma Systems Foundation

PTSF Trauma Center List

Census Tract Boundaries

Geographic boundaries for all census tracts in Philadelphia County (FIPS 42101).

Tracts: 408

Source: U.S. Census Bureau

Census TIGER/Line Files

Demographic Data

Population, racial composition, income, and poverty statistics at the census tract level.

Survey: ACS 5-Year Estimates

Source: U.S. Census Bureau

Census Data Explorer

Drive-Time Isochrones

Polygons representing areas reachable within specified drive times from each trauma center. Generated via API.

Intervals: 5, 10, 15, 20, 30 minutes

Source: OpenRouteService API

OpenRouteService

Trauma Center Verification

Cross-reference for trauma center accreditation and verification status.

Verification: ACS/COT

Source: American College of Surgeons

ACS Verified Trauma Centers

Processed Datasets

The following processed datasets are available in the GitHub repository :

File Description Format
tracts_analysis_ready.geojson Master analysis dataset with all tract-level metrics GeoJSON
tracts_bivariate_classified.geojson Tracts with 9-class bivariate classification GeoJSON
shootings_with_tracts.csv Shooting incidents with tract assignment CSV
trauma_centers_geocoded.csv Trauma center locations with coordinates CSV
trauma_desert_tracts.csv List of 18 trauma desert tracts with statistics CSV
hospital_catchment_statistics.csv Modeled catchment for each trauma center CSV
golden_hour_distribution.csv Shootings by transport time category CSV
demographic_disparity_analysis.csv Statistical comparison results CSV
oaxaca_decomposition_results.csv Oaxaca-Blinder decomposition output CSV

Data Dictionary

Complete field definitions are documented in the repository:

View Data Dictionary on GitHub

Key Fields in Master Dataset

Field Type Description
GEOID String Census tract identifier (11-digit FIPS code)
total_shootings Integer Total shooting incidents 2015-2025
annual_shootings_per_sq_mi Float Shootings per square mile per year
time_to_nearest Integer Minutes to nearest Level I trauma (5, 10, 15, 20, 30)
nearest_trauma_center String Name of nearest Level I Adult trauma center
bivariate_class Integer Classification 1-9 (9 = trauma desert)
pct_black Float Percent Black population (ACS)
pct_poverty Float Percent below poverty line (ACS)
total_population Integer Total population (ACS)
