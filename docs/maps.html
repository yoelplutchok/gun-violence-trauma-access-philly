Interactive Maps

Explore the geospatial analysis through interactive visualizations

These interactive maps allow you to explore the data in detail. Click on census tracts to view statistics, toggle layers on and off, and zoom to areas of interest.

Available Maps

Bivariate Classification Map

3x3 classification combining shooting density and transport time. Dark teal indicates trauma deserts (high violence, poor access). Click any tract to view classification, shooting counts, and transport time.

Open Interactive Map

Trauma Center Isochrones

Drive-time polygons showing 5, 10, 15, 20, and 30 minute zones from each Level I trauma center. Includes shooting incident heatmap overlay showing concentration of violence. Toggle layers using the control panel.

Open Interactive Map

Patient Flow Visualization

Lines connecting each census tract to its nearest Level I trauma center. Line thickness represents shooting volume; color represents transport time. Visualizes the modeled catchment area for each hospital.

Open Interactive Map

Map Usage Guide

Navigation

  • Zoom: Use mouse scroll wheel or +/- buttons
  • Pan: Click and drag to move the map
  • Reset view: Click the home button to return to default extent

Interactivity

  • Tract information: Click any census tract to view a popup with detailed statistics
  • Hospital information: Click trauma center markers for hospital details
  • Layer toggle: Use the layer control (upper right) to show/hide different data layers

Legend

The bivariate map uses a 3x3 color scheme:

  • Light gray: Low violence, good access (Class 1)
  • Green tones: High violence, good access (Classes 7-8)
  • Blue tones: Low violence, poor access (Classes 2-3)
  • Dark teal: High violence, poor access - Trauma Desert (Class 9)
